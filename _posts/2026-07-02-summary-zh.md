## [首个能生长分裂的合成细胞诞生](https://www.quantamagazine.org/for-the-first-time-a-cell-built-from-scratch-grows-and-divides-20260701/) ⭐️ 10.0/10

由明尼苏达大学 Kate Adamala 博士领导的研究人员创造了 SpudCell，这是首个从零构建并能生长和分裂的合成细胞。该成果于 2026 年 7 月 1 日公布，标志着合成生物学的一个重要里程碑。 这一突破表明，最小合成细胞能够执行生长和分裂等基本生命过程，使我们更接近理解生命起源，并为生物技术、医学和材料科学领域的未来应用铺平道路。同时，它也挑战了生命本身的定义。 SpudCell 缺乏细胞骨架，通过一种涉及膜重塑的独特机制进行分裂，而非天然细胞的蛋白质机器。该工作最初被《细胞》期刊拒绝，一位审稿人声称 SpudCell 不是真正的生物学，导致 Adamala 在同行评审前直接将手稿发布给记者。

hackernews · defrost · 7月1日 14:20 · [社区讨论](https://news.ycombinator.com/item?id=48747304)

**背景**: 合成生物学旨在从非生命组分构建人工细胞，以理解生命的基本原理。此前的研究已在合成细胞中实现了喂养、生长和 DNA 复制，但细胞分裂一直是一个主要障碍，因为天然分裂依赖于复杂的细胞骨架重排。Adamala 团队通过设计更简单的分裂过程绕过了这一难题。

## [Kimi K2.7 代码模型现已在 GitHub Copilot 中可用](https://github.blog/changelog/2026-07-01-kimi-k2-7-is-now-available-in-github-copilot/) ⭐️ 8.0/10

Moonshot AI 的开源智能编码模型 Kimi K2.7 Code 现已全面在 GitHub Copilot 中可用，为 GPT-5.4 mini 和 Claude 提供了一个经济高效的替代方案。 这为开发者提供了一个可信赖且具有竞争力的 AI 辅助编码选项，其性能可与 Sonnet 4.6 媲美，且成本可能更低，从而减少了对美国主要 AI 提供商的依赖。 定价为每百万输入令牌 0.95 美元，缓存命中 0.19 美元，每百万输出令牌 4.00 美元，与 Moonshot 的直接定价一致，大致与 GPT-5.4 mini 持平。

hackernews · unliftedq · 7月2日 04:32 · [社区讨论](https://news.ycombinator.com/item?id=48756602)

**背景**: GitHub Copilot 是一个 AI 驱动的编码助手，提供实时代码建议和补全。Kimi K2.7 Code 是 Moonshot AI 的开源模型，专为编码和智能体任务设计，支持 256K 上下文窗口和工具调用功能。
